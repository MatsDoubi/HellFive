Vous êtes tous en dernière année d’un cursus d’un an à la prestigieuse Academie de l’illustre Valerius CIRFAPMOP. 

Valerius est un petit gens, rondouillard et nerveux. Il an est l’auteur d’une ballade encore chantée dans toutes les grandes villes.

L’académie est à 3 heures de marche de la ville d’Eau Profonde.

Vous savez qu’une seconde année est possible pour les meilleurs élèves (et peut-être les plus riches..)
Spécialisation dans les arts scéniques et optimisations acoustique. C’est le programme de la seconde année. Chaque année seuls quelques étudiants sont acceptés dans ce programme.

Comme le répète souvent Valerius “les mots sont importants, les sots les écoutent sans toujours les comprendre, les architectes du verbe les subliment et les façonnent”
